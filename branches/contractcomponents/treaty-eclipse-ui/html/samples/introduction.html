The Eclipse Clock Example

The Eclipse clock is a sample application for treaty. It is a set of Eclipse plugins that can be installed using the following Eclipse update site:

http://treaty.googlecode.com/svn/trunk/treaty-eclipse-updatesite/site.xml

The main clock plugin adds an Eclipse view to Eclipse that shows a simple clock and a list of available date formatters. These date formatters are provided by separate plugins extending the net.java.treaty.eclipse.example.clock.dateformatter extension point. The extension point can be extended in two ways:

  1. plugins can provide an implementation of the net.java.treaty.eclipse.example.clock.DateFormatter interface

  2. plugins can provide an XML based format definition instantiating the dateformat.xsd schema
